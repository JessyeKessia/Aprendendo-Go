# 📚 Aprendendo Go (Golang)

Este repositório foi criado para registrar meus estudos na linguagem **Go (Golang)**.
Aqui estou praticando conceitos básicos da linguagem e resolvendo pequenos exercícios para entender melhor como ela funciona.

## 🚀 Objetivo

O objetivo deste repositório é:

* Aprender a sintaxe da linguagem Go
* Praticar estruturas básicas de programação
* Entender o uso de slices, funções e controle de fluxo
* Criar pequenos programas no terminal

## 🧠 Conteúdos praticados

Alguns dos conceitos que estou estudando:

* Variáveis e tipos de dados
* Estruturas de controle (`if`, `for`)
* Funções
* Slices
* Manipulação de dados
* Entrada de dados pelo terminal
* Tratamento de erros

## 📂 Estrutura

Os exercícios estão organizados em arquivos `.go` simples, geralmente contendo um único `main.go`.

## ▶️ Como executar

Para rodar um dos programas:

```bash
go run main.go
```
